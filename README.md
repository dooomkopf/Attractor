# Attractor — BTC power-law residuals as a reconstructed attractor

Delay-embedding/PCA reconstruction of the Bitcoin power-law residuals,
Nambu/Wang two-scroll modelling, and SSM analysis (SSMLearn/SSMTool).
Analysed consistently in both time clocks: linear (days) and log10(days).

Paper draft: `BTC-Attractor-Reconstruction-DRAFT.pdf`.
Code map (which script produces which figure, with arguments): `CODE-MAP.md`.
