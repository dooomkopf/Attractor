  3. Warum ein Sattel und was das physikalisch heißt                                                                                                                                              
                  
  Die SSM-Basis startet bei start_deg=2 (Tangentialitätsbedingung, SSM_res_geometry.py L43-66): kein Konstant-, kein Linearterm. Der niedrigste Term ist also                                     
                  
  W(u,v) = α u² + β uv + γ v² + (höhere Ordnungen)                                                                                                                                                
                  
  Die Hessian am Ursprung ist [[2α, β], [β, 2γ]]. Klassifikation über die Determinante:                                                                                                           
                  
  ┌───────────────┬──────────────────────────┬───────────────────────────────────────────────────────┐                                                                                            
  │     det H     │           Form           │                       Bedeutung                       │
  ├───────────────┼──────────────────────────┼───────────────────────────────────────────────────────┤                                                                                            
  │ > 0, Spur > 0 │ Schale (positiv definit) │ stabile Senke, alles wird angezogen                   │
  ├───────────────┼──────────────────────────┼───────────────────────────────────────────────────────┤
  │ > 0, Spur < 0 │ umgekehrte Schale        │ instabile Quelle                                      │                                                                                            
  ├───────────────┼──────────────────────────┼───────────────────────────────────────────────────────┤                                                                                            
  │ < 0           │ Sattel (indefinit)       │ hyperbolisch — eine stabile + eine instabile Richtung │                                                                                            
  └───────────────┴──────────────────────────┴───────────────────────────────────────────────────────┘
